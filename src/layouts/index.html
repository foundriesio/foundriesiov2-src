Linux microPlatform

The Linux microPlatform combines secure updatable firmware, the Linux kernel, a minimal distribution built using Open Embedded/Yocto, and a container-based application runtime. This enables secure remote management of all your product's software using standard tools. A subscription to the Linux microPlatform update stream gives you access to stabilized continuous updates of all these technologies, keeping you in full control of when to integrate new features and fixes.

MORE ABOUT LINUX MICROPLATFORM

Zephyr microPlatform

The Zephyr microPlatform is secure RTOS software, combining the MCUboot secure bootloader, the latest Zephyr Project software, and reference applications to provide a continuously tested, updatable, cross architecture solution for microcontroller based products.
