DEVICES AND PLATFORMS

HARDWARE

The microPlatforms are hardware and architecture independent. Foundries.io use a small number of reference hardware platforms to create CI test labs to enable automated end to end testing across volumes of devices. Our goals for reference devices include:

  • Functionality
  • Strong upstream support
  • Global availability of devices and low cost development boards

Documentation is provided on how to use the microPlatforms on any hardware supported in the relevant upstream projects (i.e. the Zephyr Project and the Linux kernel). We provide builds for several popular hardware platforms as examples, and are also working with SoC and MCU manufacturers to create a "microPlatform Certified" program.
